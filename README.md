### Краткое описание альтернативной модели (KMeans)

Для сравнения с основным решением (GMM) была реализована кластеризация методом KMeans. Использованы те же 5 признаков и предобработка. Результат на Kaggle составил 0.66851, что ниже, чем у GMM (0.84487). Это объясняется тем, что KMeans предполагает сферическую форму кластеров, тогда как реальные данные имеют более сложную структуру. Таким образом, GMM является более подходящим методом для данной задачи.
